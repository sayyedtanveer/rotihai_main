DB_URL="$DATABASE_URL"

TROUBLESHOOTING:

Problem: Still see fewer tables after migration
Solution: 
  1. Check for errors: psql "$DB_URL" -f FULL_DATABASE_MIGRATION.sql 2>&1 | grep -i error
  2. Verify specific table: psql "$DB_URL" -c "\d orders"
  3. Rerun migration: psql "$DB_URL" -f FULL_DATABASE_MIGRATION.sql

Problem: Connection timeout
Solution:
  1. Verify host is correct: psql -h your_host -c "SELECT 1"
  2. Check IP whitelist in Neon console
  3. Ensure SSL if required: ?sslmode=require

Problem: Permission denied
Solution:
  1. Check user role: psql "$DB_URL" -c "SELECT current_user;"
  2. Grant permissions if needed
  3. Use correct DATABASE_URL
